# Festin UI 디자인 컨셉

## 🎨 디자인 방향

### 컨셉: "Modern Festival Vibes"
- **분위기**: 축제의 화려함 + 테이블링의 세련됨
- **키워드**: 미니멀, 다크모드, 네온사인 느낌, 그라데이션 포인트

---

## 🎯 색상 팔레트

### 베이스 컬러 (다크 모드)
```
배경: #0a0a0a (순수 블랙에 가까운 다크)
서페이스: #1a1a1a (카드, 패널)
서페이스 호버: #262626
테두리: #2a2a2a (은은한 경계선)
```

### 포인트 컬러 (그라데이션)
```
메인: #8b5cf6 → #ec4899 (보라-핑크 그라데이션)
- 버튼, 중요한 액션
- 브랜딩 요소
- 강조 텍스트
```

### 텍스트 컬러
```
기본: #ffffff (순백)
보조: #a3a3a3 (회색)
비활성: #525252 (어두운 회색)
```

## 📱 화면별 디자인 컨셉

### 1. 첫 화면 (랜딩/브랜딩)
```
┌─────────────────────────────┐
│                             │
│                             │
│         [FESTIN 로고]        │  ← 큰 그라데이션 텍스트
│      축제의 새로운 경험        │  ← 작은 설명
│                             │
│                             │
│     [로그인하고 시작하기]      │  ← 그라데이션 버튼
│                             │
│                             │
│   스태프이신가요? 여기를 클릭   │  ← 작은 텍스트 링크
│                             │
└─────────────────────────────┘
```

**특징:**
- 중앙 정렬, 미니멀
- 이모지 없음
- 그라데이션 로고만 강조
- 스태프는 하단에 작게

---

### 2. 로그인 화면 (방문자용)

┌─────────────────────────────┐
│  ← Festin                   │  ← 헤더

│     축제 부스 대기 등록       │

- 역할 선택 없음 (기본 방문자)

- 큰 버튼

## 🎯 컴포넌트 스타일

### Button
```css
/* Primary (그라데이션) */
background: linear-gradient(135deg, #8b5cf6, #ec4899);
padding: 12px 24px;
border-radius: 8px;
font-weight: 600;

/* Secondary (블랙 아웃라인) */
background: transparent;
border: 1px solid #2a2a2a;
color: #ffffff;

/* Danger */
background: #ef4444;
```

background: #1a1a1a;

## 🎭 아이콘 사용

이모지 대신 심플한 아이콘 또는 인디케이터:
- 운영중: 녹색 원(●)
- 마감: 회색 원(○)
- 상태: 컬러 바 또는 점
- 액션: 화살표(→, ←, ↓)

## 📱 레퍼런스

**영감을 받은 앱:**
- 테이블링: 블랙 베이스, 미니멀
- 트래블러: 그라데이션 포인트
- 토스: 깔끔한 타이포

**Festin만의 차별점:**
- 축제 느낌의 그라데이션 (보라-핑크)
- 네온사인 느낌의 포인트 컬러
- 미니멀하지만 화려한 브랜딩
